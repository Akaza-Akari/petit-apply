## Configure
Edit `config.ini.example` and rename to `config.ini`, or make `config.ini` and edit this.
```ini
[db]
host = "localhost"
user = "root"
pass = "your-awesome-pass"
name = "your-awesome-db"
table = "your-awesome-table"
```
