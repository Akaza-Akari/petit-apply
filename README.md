## Configure
Make `config.ini` and edit this.
```ini
[root]
domain = "example.com"
api = "https://api.example.com"
[db]
host = "localhost"
user = "root"
pass = "your-awesome-pass"
name = "your-awesome-db"
table = "your-awesome-table"
```
